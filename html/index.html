Aplicativos com Suporte

Blender

O Blender é um software de criação 3D gratuito, com um grande número de versões como os renderizadores Cycles CPU, Cycles GPU e EEVEE.

Maxon Cinema

Cinema 4D é um dos softwares 3D mais populares, usado na área de motion graphics, design arquitetônico, visualização de produto, VFX e muitos mais.

Autodesk 3ds Max

O Autodesk 3ds Max é um software de computação gráfica 3D usado nas áreas de arquitetura, mídia, VFX e outras indústrias.

Autodesk Maya

O Autodesk Maya é o software 3D líder para animações, séries de TV, videogame, efeitos visuais, entre outros.
